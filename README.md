# buster_the_rover

About Buster: 
Arduino based robot utilizing a set of sensors to achieve indoor autonomous driving.

Hardware:

- Board:
    Arduino Uno
    
- Sensors:
    - HCSR04 Ultrasonic Sensor
    - Start/Stop Buttons
    - Two Momentary collision detection buttons
    
- Motors/Drivers:
    - 4x small geared DC motors
    - Adafruit motor shield v2
    - 1x servo
